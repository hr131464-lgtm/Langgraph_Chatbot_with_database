# LangGraph Chatbot with Database

An AI-powered chatbot built using LangGraph, LangChain, Groq LLM, and Streamlit with database integration for storing conversation history.  
This project demonstrates how to create a stateful conversational AI application with memory, workflow orchestration, and an interactive web interface.

---

##  Features

-  AI chatbot powered by Groq LLM
-  Stateful conversation flow using LangGraph
-  Database integration for chat history storage
-  Interactive UI with Streamlit
-  Fast response generation
-  Context-aware conversations
-  Secure API key management using `.env`
-  Modular and beginner-friendly project structure

---

##  Tech Stack

| Technology | Purpose |

#  How the Chatbot Works

1. User enters a message through the Streamlit interface  
2. LangGraph manages the conversational workflow  
3. LangChain processes prompts and interactions  
4. Groq LLM generates AI responses  
5. Database stores conversation history  
6. The response is displayed back to the user
